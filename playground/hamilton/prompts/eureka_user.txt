## 当前最佳表达式（Current Best）
- `insight.md` 顶部包含一个“Current Best（auto-updated）”区块：它是当前系统认为最可信的表达式。
- 你本轮要做的关键决策之一：是否更新 Current Best（`update_best`）。

## 可复用工具库（推荐，非必须）
- `skills/eurekatool/tool.py`（stdlib-only）
- 你可以在脚本里直接 `from skills.eurekatool import tool`

## 本轮职责（简化）
- 读取 `experiment.json` 本轮结果（`rounds[{input_data[round]}]`）和当前 `analysis.md` 历史。
- 在 `data.csv` 上做一次基础误差/残差评估（最小但可复现）。
- 写 `insight.md`：本轮结论 + 是否更新 Current Best + 下一轮建议（可执行、可验证）。
- 不要求固定模板化步骤，重点是有结论、有证据、可复现。

## 输出边界
- 只追加，不覆盖 `analysis.md`、`insight.md`。
- 不改 `experiment.json` 结构（保持系统自动写入）。

## 常用终端命令（可直接用）
1) 追加 insight
```bash
cat >> insight.md <<'EOF'
## Round {input_data[round]}
- ...
EOF
```
2) 本轮脚本
```bash
cat > history/round{input_data[round]}/scripts/round{input_data[round]}_residual_report.py <<'PY'
print("report")
PY
python3 history/round{input_data[round]}/scripts/round{input_data[round]}_residual_report.py
```

3) 可选：生成基础报告块（你再补充决策与结构化信号）
```bash
python3 skills/eurekatool/scripts/round_report.py --round {input_data[round]} --recompute
```
