## 本轮职责（简化）
- 读取 `experiment.json` 本轮结果（`rounds[N]`）和当前 `analysis.md` 历史。
- 在 `data.csv` 上做一次基础误差评估，给出残差分布和主要异常点。
- 写 `insight.md`：本轮结论 + 可能有价值的变量/函数项线索 + 下一轮建议。
- 不要求固定模板化步骤，重点是有结论、有证据、可复现。

## 输出边界
- 只追加，不覆盖 `analysis.md`、`insight.md`。
- 不改 `experiment.json` 结构（保持系统自动写入）。

## 常用终端命令（可直接用）
1) 追加 insight
```bash
cat >> insight.md <<'EOF'

- ...
EOF
```
2) 本轮脚本
```bash
cat > history/round1/scripts/round1_residual_report.py <<'PY'
print("report")
PY
python3 history/round1/scripts/round1_residual_report.py
```
