# VIV 方程族发现

## 背景

涡激振动（Vortex-Induced Vibration, VIV）是流固耦合中的经典现象：气流经过弹性支撑的钝体结构时，周期性涡脱落可以激发结构的自持振动。

本数据来自一座大跨度悬索桥节段模型的风洞实验。实验在 **5 个不同风速** 下进行，均处于 VIV 锁定区间内（涡脱频率与结构固有频率锁定）。每个风速下，系统呈现稳定的极限环振动，但稳态振幅随风速不同而变化。

## 任务

从 5 组不同风速的振动时序数据中，发现控制该系统的常微分方程族：

```
x'' = f(x, x'; U)
```

其中 `U` 为风速。对每个风速，发现对应的 `f`，使得 `a ≈ f(x, v)`。

**核心挑战**：这 5 组数据来自同一物理结构，只是风速不同。因此期望发现的方程族具有 **相同的函数结构**（相同的项），只有 **系数随风速变化**。这一结构一致性是衡量发现质量的关键指标。

## 数据说明

### 5 个风速的配对数据

每个风速有两组实验（不同初始条件，同一物理系统）：

| 风速 | 训练集（Z2S：零→稳态） | 测试集（B2S：大振幅→稳态） |
|------|------------------------|---------------------------|
| 2.48 m/s | `input/U248_train.csv` | `input/U248_test.csv` |
| 2.54 m/s | `input/U254_train.csv` | `input/U254_test.csv` |
| 2.60 m/s | `input/U260_train.csv` | `input/U260_test.csv` |
| 2.73 m/s | `input/U273_train.csv` | `input/U273_test.csv` |
| 2.82 m/s | `input/U282_train.csv` | `input/U282_test.csv` |

- **训练集（Z2S = zero-to-stable）**：零初始位移，系统从静止出发，振幅逐渐增长到稳态极限环
- **测试集（B2S = big-to-stable）**：初始位移约 15 mm（大于稳态振幅），振幅从大振幅衰减到稳态极限环
- 同一风速下，训练和测试的稳态振幅应一致

### 列定义

每个 CSV 文件都包含相同的 4 列：

| 列 | 含义 | 单位 |
|----|------|------|
| t | 时间 | s |
| x | 位移 | mm |
| v | 速度 dx/dt | mm/s |
| a | 加速度 d²x/dt² | mm/s² |

- 采样率 100 Hz（原始 300 Hz 降采样），每文件约 **20000–30000 行 × 4 列**
- `v` 和 `a` 通过中心有限差分从位移序列计算
- 首尾各裁剪 50 点以避免差分边界伪影

### Bonus OOD 数据（可选验证）

| 文件 | 工况 | 说明 |
|------|------|------|
| `input/U000_free_vibration.csv` | 无风 | 纯自由振动衰减（无气动力输入） |
| `input/U216_below_lockin.csv` | 2.16 m/s | 锁定区间以下（振幅是否受抑制？） |
| `input/U300_above_lockin.csv` | 3.00 m/s | 锁定区间以上（系统行为变化？） |

这些工况没有配对训练集，仅作为额外泛化验证。

## 物理先验（可选参考，非约束）

1. **弹性恢复力**：结构有弹簧支撑，必然存在与位移相关的恢复力项（线性弹簧项是主导项）
2. **阻尼与自激**：
   - 小振幅：负阻尼（能量从气流注入结构，振幅增长）
   - 大振幅：正阻尼（能量耗散超过注入，振幅衰减）
   - 平衡点即为极限环振幅
3. **极限环**：训练和测试从不同方向收敛到相同稳态振幅，确认系统存在稳定极限环
4. **振动频率**：约 2 Hz（可从任意数据集直接观测，各风速间略有差异）
5. **风速效应**：风速改变的主要是气动力（阻尼/自激项的强度），弹性恢复力（弹簧）基本不变

## 方法提示

本问题涉及含交叉项的非线性 ODE，PySR 需要：
- **精确的模板设计**（TemplateExpressionSpec）：基于物理假设约束搜索空间，而非暴力全空间搜索
- **充足的搜索迭代**：推荐 `niterations=5000`（或更多），**不要使用 `timeout_in_seconds`**。用迭代次数控制搜索深度，通过观察 Pareto 前沿是否收敛来判断是否需要继续
- **早停机制**：可设置 `early_stop_condition` 在达到目标精度时提前退出，例如 `"f(loss, complexity) = (loss < 1e-3) && (complexity < 15)"`。这比超时更合理——搜索到位就停，没到位就继续跑满迭代

## 数据使用规则（防止数据泄露）

- **训练数据**（可用于拟合方程）：仅 `U{speed}_train.csv`（Z2S 工况）的 (x, v, a) 三元组
- **测试数据**（仅用于最终评估，不可用于模型选择或调参）：`U{speed}_test.csv`（B2S 工况）
- **OOD 数据**（仅用于最终评估）：`U000_free_vibration.csv`、`U216_below_lockin.csv`、`U300_above_lockin.csv`
- 模型选择和超参数调优只能基于训练集的交叉验证或训练集指标

## 评估标准

**核心理念**：对非线性振子，逐点加速度误差小不代表长期演化正确。传统方法即使在瞬时加速度上拟合很好，也可能在时间积分后给出完全错误的轨迹。因此，本任务的首要目标不是最小化 MSE，而是**发现能正确复现动力学行为的方程**。

### 1. 极限环复现（首要指标）

将发现的 ODE 用 `scipy.integrate.solve_ivp` 积分，验证是否能复现正确的极限环行为：

**a) 稳态振幅误差**（核心定量指标）

对每个风速，分别从两种初始条件积分到稳态（至少 200s），提取稳态振幅 A_pred，与实验值对比：

| 风速 | 实验稳态振幅 A_exp (mm) |
|------|------------------------|
| 2.48 m/s | 11.1 |
| 2.54 m/s | 11.1 |
| 2.60 m/s | 10.0 |
| 2.73 m/s | 8.7 |
| 2.82 m/s | 7.1 |

- 从训练初始条件（x₀ ≈ 1 mm, v₀ = 0）出发：振幅应增长到 A_exp
- 从测试初始条件（x₀ ≈ 14 mm, v₀ = 0）出发：振幅应衰减到 A_exp
- 指标：`amplitude_error = |A_pred - A_exp| / A_exp`（相对误差，越小越好）
- 两种初始条件应收敛到相同的稳态振幅（极限环的本质特征）

**b) 定性行为检查**
- Z2S 条件：轨迹是否先增长后饱和？（不能单调衰减或发散）
- B2S 条件：轨迹是否先衰减后稳定？（不能持续增长或发散）
- 两个方向是否收敛到同一振幅？

### 2. 结构一致性（跨风速）

- 5 个方程是否具有**相同的函数结构**（相同的项）？这是同一物理系统在不同参数下的必然要求
- 系数随风速的变化是否合理？（单调或有物理可解释的趋势）
- 恢复力相关系数（与 x 有关的项）应跨风速基本不变（弹簧不变）
- 阻尼/自激相关系数（与 x' 有关的项）可以随风速变化（气动力变化）
- 如果某个风速出现了其他风速没有的额外项，需要解释原因

### 3. 简洁性与可解释性

- 在动力学行为同等正确的前提下，项数越少越好
- **每个项必须有明确的物理解释**——不接受"数值上拟合好但不知道为什么"的黑箱项。对发现的方程中的每一项，必须回答：
  - 这个项代表什么物理机制？（恢复力？阻尼？自激？非线性耦合？）
  - 它的符号和量级是否符合物理预期？
  - 去掉它会导致什么动力学行为变化？
- 如果发现了意想不到的项（比如交叉项 x²v、高次项等），这可能是**真正有价值的物理发现**——要在 findings.md 中详细分析其物理含义

### 4. 加速度拟合（诊断指标，非优化目标）

在训练集上报告 `R² = 1 - Σ(a_pred - a_true)² / Σ(a_true - ā)²`，用于诊断方程的瞬时精度。

注意：这个指标仅作为参考。R² 高但轨迹积分不收敛的方程是失败的；R² 一般但轨迹行为正确的方程可能是更好的发现。

### 5. 物理一致性与发现

- 方程是否包含恢复力项（与 x 相关）？
- 方程是否能解释自激机制（小振幅负阻尼 + 大振幅正阻尼 → 极限环）？
- 无风工况（U000）：方程是否退化为纯阻尼振子？（仅衰减，无自激）
- 锁定区间外（U216, U300）：行为是否合理？
- **系数随风速的变化规律**：能否拟合出系数与风速的函数关系？这本身就是一个物理发现

## 发现记录要求

在 `findings.md` 中，对每个发现的方程，必须包含：

1. **方程表达式**及各项的物理解释
2. **系数表**（跨风速），标注哪些系数基本不变（结构属性）、哪些随风速变化（气动属性）
3. **物理洞察**：方程揭示了什么物理规律？比如：
   - 非线性阻尼的形式说明了什么气动力机制？
   - 极限环振幅与风速的关系能否从方程系数推导出来？
   - 是否存在经典文献中未报道过的项？
4. **消融分析**：逐个去掉非线性项，观察动力学变化，证明每个项的必要性

## Baseline 参照（EvLOWN 论文结果）

EvLOWN 在此数据上发现的方程族结构为：

```
x'' + ω²x + c₁x³ + c₂x' + c₃x'³ + c₄x'⁵ = 0
```

其中 ω² ≈ 163，5 项跨风速结构一致。EvLOWN 未报告逐点 MSE，其优势在于：
- 结构一致性好（5 个风速共享相同函数形式）
- 动力学行为定性正确（训练/测试轨迹均能收敛到极限环）
- 消融实验表明弱非线性项（x'³, x'⁵）虽系数极小，但对长期演化至关重要

"比 EvLOWN 更好"意味着在以下一项或多项上有提升：
1. 稳态振幅更精确（amplitude_error 更小）
2. 跨初始条件泛化更鲁棒（Z2S 和 B2S 都收敛到正确振幅）
3. 方程更简洁（更少的项）
4. 系数随风速的变化趋势更平滑、更有物理解释力
